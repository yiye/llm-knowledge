# Cloudflare Tunnel + Zero Trust 安全部署

## 💡 使用场景

在服务器或 Mac Mini 部署内网服务（如 Clawdbot）时的安全方案

## 🔐 方案要点

### 1. Cloudflare Tunnel
- **作用**：让内网服务可以在外部访问
- **优势**：无需开放防火墙端口，不暴露服务器 IP

### 2. Cloudflare Zero Trust Access
- **作用**：为服务添加身份验证层
- **优势**：防止未授权访问

## ✅ 核心优势

- 🆓 **完全免费**
- 🔒 **安全性提升**：双重保护（隧道 + 身份验证）
- 🚀 **易于配置**：Cloudflare 控制台操作

## 📌 适用服务

- Clawdbot（Claude 机器人）
- 其他需要外部访问的内网服务
